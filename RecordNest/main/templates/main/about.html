Sobre Nosotros

Este proyecto es un Trabajo de Fin de Grado para el Grado en Ingeniería del Software de la Universidad de Sevilla desarrollado por Héctor Noguera González

Su objetivo es ofrecer una aplicación web que permita a los usuarios organizar, explorar y mantener su propia colección musical de manera sencilla e intuitiva.

Todos los datos de álbumes, portadas, géneros, artistas y metadatos han sido obtenidos gracias a la API de Discogs y la API de Deezer.

Agradezco a estas plataformas por facilitar el acceso a sus servicios y contribuir al desarrollo de este proyecto.
